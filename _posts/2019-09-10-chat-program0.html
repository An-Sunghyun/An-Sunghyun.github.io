[JAVA] Chat program using socket communication

It is a chat program based on Java and it is implemented by utilizing all basic Java technology such as linking database, community and so on.

Main Frame

On the main screen, you can chat with other users, and it also provides functions such as changing the color of the background screen and checking current contact list.

Class diagram

This class diagram shows how Java classes within the program are linked.

Database

The database basically has member information. Through this, members' personal information such as name, ID, password will be managed, and Java will link them through certain Java classes..

It implements basic community functions such as genuine registration, login, and membership.

Setting Background Color

On this frame, you can select background via file chooser or just apply random color.

If you choose random color, the color of background will change randomly.

Checking Current Users

On this screen, you can see the list of users currently logged on.

Originally, I wanted to add whispering functions by linking them, but I was not able to implement them due to lack of capability.
