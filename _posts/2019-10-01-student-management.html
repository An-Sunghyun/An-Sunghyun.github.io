[JAVA] 학생 관리 프로그램

자바 기반 프로그램으로 파일 선택, 로그인 등 자바의 기본적인 기술을 활용하여 구현함

Main Frame

메인 화면에서는 학생들의 조건을 선택할 수 있음

조건은 체크 박스와 트리 형태로 구성되어 있으며 조건에 맞는 결과가 실시간으로 출력됨

When Checked

위에서 볼 수 있듯이, 체크 박스를 선택하면 조건에 맞는 결과만 필터링되어 화면에 출력됨

Login Frame

위 화면은 미리 저장된 계정과 비밀번호를 입력하여 프로그램을 사용할 수 있는 로그인 화면임

본 프로그램에서는 데이터베이스를 사용하지 않았기 때문에 일회용 계정을 매번 추가해야 함

File Chooser Frame

데이터베이스는 사용하지 않았으나 파일 입출력을 활용하여 텍스트 파일을 통해 데이터를 관리할 수 있음

따라서 파일 선택기를 통해 데이터를 쉽게 검색하고 사용할 수 있음. 물론 연동되어 자동으로 데이터를 읽고 쓸 수 있는 데이터베이스보다는 편의성이 부족함
