[JAVA] Student Management Program

It is a rhythm game based on Java and it is implemented by utilizing all basic Java technology.

Main Frame

On the main screen, you can select student conditions.

The condition consists of check boxes and a tree, and the results that meet the condition are printed in real time.

When Checked

As you can see, when you check the check box, only the results that meet the conditions are displayed on the screen.

Login Frame

This screen is a login screen that allows users to use the program by entering their pre-saved account and password.

Because the database is not used, the administrator must add user information each time.

File Chooser Frame

The database was not used, but the ability to read data through a text file is enabled.

Therefore, data can be easily retrieved and used through the file chooser.
